# 博客网站需求文档

## 项目概述

### 项目名称
个人博客网站系统 (Personal Blog System)

### 项目描述
基于Java技术栈开发的现代化博客网站，提供完整的博客发布、管理和互动功能。适合Java实习生学习和实践Web开发技能。

### 目标用户
- 博客作者：发布和管理博客内容
- 普通访客：浏览和互动博客内容
- 管理员：系统管理和维护

## 技术栈选择

### 后端技术栈
- **框架**: Spring Boot 2.7.x
- **数据库**: MySQL 8.0
- **ORM**: MyBatis Plus
- **缓存**: Redis
- **安全**: Spring Security + JWT
- **文档**: Swagger/OpenAPI 3
- **构建工具**: Maven
- **Java版本**: JDK 8/11

### 前端技术栈
- **框架**: Vue.js 3.x
- **UI组件库**: Element Plus
- **状态管理**: Vuex/Pinia
- **路由**: Vue Router
- **HTTP客户端**: Axios
- **构建工具**: Vite

### 部署和运维
- **服务器**: 阿里云ECS
- **Web服务器**: Nginx
- **数据库**: MySQL 8.0
- **CI/CD**: 手动部署或简单脚本

## 非功能性需求

### 1. 性能要求
- 页面加载时间 < 3秒
- 支持并发用户数 > 100
- 数据库查询优化
- 静态资源CDN加速

### 2. 安全要求
- SQL注入防护
- XSS攻击防护
- CSRF攻击防护
- 密码加密存储
- 敏感信息脱敏

### 3. 可用性要求
- 响应式设计，支持移动端
- 友好的用户界面
- 良好的用户体验
- 错误提示清晰

### 4. 可维护性
- 代码规范统一
- 充分的注释文档
- 模块化设计
- 日志记录完善

## 数据库设计

### 主要数据表

#### 用户表 (users)
```sql
- id (主键)
- username (用户名)
- email (邮箱)
- password (密码)
- nickname (昵称)
- avatar (头像)
- status (状态)
- create_time (创建时间)
- update_time (更新时间)
```

#### 文章表 (articles)
```sql
- id (主键)
- title (标题)
- content (内容)
- summary (摘要)
- cover_image (封面图)
- category_id (分类ID)
- author_id (作者ID)
- status (状态)
- view_count (浏览量)
- like_count (点赞数)
- create_time (创建时间)
- update_time (更新时间)
```

#### 分类表 (categories)
```sql
- id (主键)
- name (分类名)
- description (描述)
- parent_id (父分类ID)
- sort_order (排序)
- create_time (创建时间)
```

#### 标签表 (tags)
```sql
- id (主键)
- name (标签名)
- color (颜色)
- create_time (创建时间)
```

#### 文章标签关联表 (article_tags)
```sql
- article_id (文章ID)
- tag_id (标签ID)
```

#### 评论表 (comments)
```sql
- id (主键)
- article_id (文章ID)
- user_id (用户ID)
- parent_id (父评论ID)
- content (评论内容)
- status (状态)
- create_time (创建时间)
```

## 项目结构

```
blog-backend/
├── src/main/java/com/blog/

```

```
blog-frontend/
├── src/

```

## 开发计划

### 第一阶段：基础框架搭建（1-2周）
1. 项目初始化
2. 数据库设计和创建
3. 基础框架配置
4. 用户认证系统

### 第二阶段：核心功能开发（2-3周）
1. 文章管理功能
2. 分类标签管理
3. 基础前端页面

### 第三阶段：互动功能开发（1-2周）
1. 评论系统
2. 点赞收藏功能
3. 搜索功能

### 第四阶段：完善和优化（1周）
1. 管理后台
2. 性能优化
3. 安全加固
4. 测试和部署

## 学习收益

通过这个项目，你将学习到：

1. **Spring Boot框架**：现代Java Web开发的主流框架
2. **数据库设计**：关系型数据库设计和优化
3. **前后端分离**：RESTful API设计和前后端协作
4. **安全认证**：JWT Token和Spring Security
5. **缓存技术**：Redis缓存应用
6. **部署运维**：Linux服务器部署和Nginx配置

这个项目涵盖了Web开发的各个方面，非常适合Java实习生提升技能和积累项目经验。
